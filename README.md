# Project : Hulu Cone

### Tech stack and Features

- Nextjs ( React js )
- Fetching data from TMDB API with Server side rendering
- My first data fetch from an External API
- Fully Responsive
- Working on More features.

<!-- Run it locally -->

## Run it locally.

```bash
 git clone https://github.com/Montu-Gohain/hulu-clone.git
 yarn install
 yarn run dev
```
